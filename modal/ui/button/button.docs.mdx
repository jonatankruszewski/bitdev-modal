## A button component that receives a function to get launched onClick

### Component usage

importance: 'primary' | 'secondary'
text: a string to be rendered
onClick: function to launch onClick

```js live
<Button
  importance='primary'
  text='hello from Button'
  onClick={() => alert("Hello World")}
/>
```
