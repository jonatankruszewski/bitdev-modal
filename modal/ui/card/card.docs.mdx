## React Component for rendering a basic card.
Accepts a title, subtitle and a paragraph text, as well as children.

### Component usage

```ts live
<Card
  title='React is awesome'
  subTitle='Also TypeScript'
  text='Hello from Card'
>
<div>Hello I am a child</div>
</Card>
```
